Bingo Game

A Bingo game uses HTML to build the 5x5 grid for the player's card and CSS to style the board and highlight marked numbers. JavaScript generates the random numbers on the card, "calls" new numbers one by one, and checks the player's card for a winning "BINGO" pattern.
